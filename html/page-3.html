KISS CSS Guidelines (Keep It Simple, Silly!)

Writing Clean, Organized, and Minimal Code

1. Keep it Organized

  • Use clear naming for classes (e.g., .btn-primary instead of .bp).
  • Group related styles together.
  • Use comments to separate sections.

2. Stick to a Naming Convention

Use BEM (Block, Element, Modifier) or a simple, readable convention

6. Avoid !important

The !important rule in CSS is used to add more importance to a property/value than normal. In fact, if you use the !important rule, it will override ALL previous styling rules for that specific property on that element!

7. Keep It Responsive

Use relative units (em, rem, %) instead of fixed pixels.

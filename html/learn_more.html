KISS HTML Guidelines (Keep It Simple, Silly!)

Writing Clean, Organized, and Minimal Code

1. Always start with:

This keeps your code clean and readable.

2. Close Your Tags

✅ Good:

<p>Hello!</p>

❌ Bad:

<p>Hello!

(Forgets the closing tag)

3. Use Semantic Elements:

Instead of <div> for everything, use:
< header > for the top section
< nav > for menus
< section > for main content
< footer > for bottom info
